---

## 📱 推荐方案：Google Sheets

### 为什么选Google Sheets？

| 优点 | 说明 |

| ✅ 免费 | Google账号即可 |

## 🚀 快速开始（5分钟）

### 第一步：导入模板

1. 打开 [Google Sheets](https://sheets.google.com)
2. 新建空白表格
3. 文件 → 导入 → 上传
4. 选择 `追踪表模板-导入用.csv`
5. 导入设置选"替换当前工作表"

### 第二步：设置条件格式（自动变色）

选中"判断"列，设置条件格式：

| 条件 | 格式 |
|------|------|
| 包含"加仓" | 🟢 绿色背景 |
| 包含"观望" | 🟡 黄色背景 |
| 包含"减仓" | 🔴 红色背景 |

### 第三步：添加到手机

1. 下载 Google Sheets App
2. 登录同一账号
3. 表格会自动同步

## 🎯 判断标准（设为公式更好）

### 各指标阈值

| 指标 | 🟢 利多 | 🔴 利空 |
|------|---------|---------|
| 金银比 | < 50 | > 80 |
| COMEX库存 | > 150M | < 120M |
| 动量分数 | > 70 | < 30 |
| COT商业 | < -50,000 | > -20,000 |

### 汇总判断

| 利多数 | 利空数 | 判断 |
|:------:|:------:|------|
| ≥4 | ≤1 | 加仓 |
| 3 | 2 | 观望 |
| 2 | 3 | 准备减仓 |
| ≤1 | ≥4 | 减仓 |

## 📱 每日操作流程

### 早上（开盘前）2分钟

```
1. 打开 silverdata.io
2. 打开 Google Sheets（手机App）
3. 填入：白银价格、黄金价格、金银比、动量
4. 完成
```

### 周三额外（收盘后）3分钟

```
1. 点击 silverdata.io → Inventories
2. 填入：COMEX库存（Registered）
3. 填入：SLV持仓、PSLV持仓
4. 对比上周，判断趋势
```

### 周五额外（晚上）3分钟

```
1. 点击 silverdata.io → COT Analysis
2. 填入：COT商业、COT投机
3. 看商业状态是否REDUCED SHORT
```

### 每天收盘后（1分钟）

```
1. 数利多/利空信号
2. 填判断：加仓/观望/减仓
3. 填今天的操作（如果有）
```

## 🔧 高级技巧：设置公式

### 自动判断利多/利空

在"利多数"列输入公式（假设数据在B-I列）：

```
=COUNTIF(B2:I2,"利多")+IF(C2<50,1,0)+IF(E2>70,1,0)+IF(F2>150,1,0)
```

### 自动判断操作建议

```
=IF(J2>=4,"加仓",IF(K2>=4,"减仓","观望"))
```

### 条件格式自动变色

选中库存列，添加条件格式：
- < 120 → 红色
- 120-150 → 黄色
- > 150 → 绿色
