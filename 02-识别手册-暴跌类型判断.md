# 识别手册：交割驱动暴跌 vs 普通市场面暴跌

> 一眼区分两种暴跌，知道什么时候必须跑、什么时候可以扛

---

## 一、终极判断口诀（背下来够用）

### 交割驱动暴跌（必须减仓）
> **白银领跌、倍数放大（2-3倍）、库存破线、交割窗口期**

### 市场面暴跌（可控，不必慌砍）
> **消息驱动、同步小跌（1-1.2倍）、库存安全、非交割窗口**

---

## 二、6条硬区分标准

### 标准1：看【谁先跌、跌几倍】（最准、最快）

| 类型 | 特征 | 真实案例（已验证） |
|------|------|-------------------|
| **交割驱动** | 白银先大跌，黄金晚几小时~次日跟跌<br>白银跌幅 ≈ 黄金的**2~3倍** | 2026.1.30 白银-17.8%，黄金-8.26%（**2.16倍**） |
| **市场面** | 金银几乎同时跌，不分先后<br>白银跌幅 ≈ 黄金的**1~1.2倍** | 普通回调时金银同步波动 |

---

### 标准2：看【日期是不是落在"高危时间窗"】

| 类型 | 特征 |
|------|------|
| **交割驱动** | 100%落在：<br>• 白银主力月：3、5、7、9、12月<br>• 精确区间：第一通知日前3天 → 最后交易日 |
| **市场面** | 出现在非主力月、非交割关键日 |

**2026年1月30日验证**：正好是2月合约第一通知日 ✅

---

### 标准3：看【COMEX白银注册库存】（决定杀伤力）

| 类型 | 库存水平 | 案例 |
|------|----------|------|
| **交割驱动** | **< 1.2亿盎司**（警戒线）<br>库存越靠近/越跌破，跌得越狠 | 2026.1月 库存1.12亿 → 暴跌-17.8% |
| **市场面** | **≥ 1.5亿盎司**，处于安全区 | 库存充足时回调幅度可控 |

---

### 标准4：看【跌的"理由"】（消息面辅助验证）

| 类型 | 特征 | 典型案例 |
|------|------|----------|
| **交割驱动** | **没有重大突发新闻**就跳水，或消息只是导火索<br>媒体说"期货平仓、交割担忧、流动性收紧" | 2026.1.30 沃什提名+交割压力叠加 |
| **市场面** | **能找到明确消息**：美联储讲话、通胀数据、美元大涨、技术破位 | 纯消息驱动的技术回调 |

---

### 标准5：看【盘口特征：有没有"踩踏式无量跌"】

| 类型 | 特征 |
|------|------|
| **交割驱动** | 成交量没特别放大，但价格**直线下砸、反弹无力**<br>本质：投机多头集体逃命，没人接盘 |
| **市场面** | 通常**放量下跌**，有买有卖<br>跌完会有反复、有反弹 |

---

### 标准6：看【跌完之后怎么走】（回头验证）

| 类型 | 特征 |
|------|------|
| **交割驱动** | 要等到**最后交易日过去+库存不再下降**才企稳<br>中间抄底大概率再吃一跌 |
| **市场面** | 消息消化完、技术跌到位，**1~3天**就可能企稳反弹 |

## 四、快速判断清单

把任意一天金银行情填进去，自己就能判断：

| 检查项 | 你的答案 | 交割特征 | 市场面特征 |
|--------|----------|----------|------------|
| 白银跌幅 ÷ 黄金跌幅 = ? | | ≥2 | ≤1.2 |
| 在3/5/7/9/12月 + 第一通知日前3天~最后交易日？ | | 是 | 否 |
| 白银注册库存 < 1.2亿盎司？ | | 是 | 否 |
| 有明确的宏观/数据/新闻利空？ | | 否或叠加 | 是（主因） |
| 是白银先跌，还是金银一起跌？ | | 白银先跌 | 一起跌 |

**判定规则**：
- ≥3条符合"交割特征" → 按交割避险执行
- ≥3条符合"市场面特征" → 按普通波动处理

## 五、对应操作动作

### 判定为：【交割驱动暴跌】

| 动作 | 说明 |
|------|------|
| 不抄底 | 交割期抄底大概率再吃一跌 |
| 不加仓 | 等待交割结束 |
| 不扛单 | 按日历执行减仓 |
| 企稳信号 | 必须等到**最后交易日过去 + 库存不再下降** |

### 判定为：【市场面暴跌】

| 动作 | 说明 |
|------|------|
| 不必恐慌砍仓 | 风险可控 |
| 可小仓位分批接 | 如果你本来想长期拿 |
| 控制仓位 | 别一把满仓 |
| 企稳信号 | 消息消化完，1-3天通常企稳 |

---

## 六、一句话总结

> **白银领跌2-3倍 + 交割窗口 + 库存破线 = 必须跑**
> 
> **金银同步跌 + 非交割期 + 有明确消息 = 可以扛**
