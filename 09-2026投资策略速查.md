## 三、1月30日被套了怎么办？（新增）

> **适用人群**：在1月30日暴跌时没减仓，目前被套住的投资者

### 当前状况评估（2026年1月31日）

| 指标 | 数值 | 说明 |

| 白银暴跌 | -27%（$120→$85） | 史上最大单日跌幅之一 |
| 黄金暴跌 | -8%（$5,520→$5,064） | 较大回调 |
| 库存 | 1.12亿盎司 | ⚠️ 仍在警戒线下 |
| 下一危险期 | 3月17-20日 | 还有45天 |

### 分析师观点（已验证）

| 机构 | 观点 | 目标价 |
|------|------|--------|
| **花旗银行** | 看好反弹 | $150（3个月内） |

| **JPMorgan前主管** | ⚠️ 警告 | 可能跌50% |

### 🚨 被套者的三种选择

#### 选择A：立刻止损（推荐给风险厌恶者）

```
适合人群：
- 仓位>总资产15%
- 无法承受再跌20%
- 近期需要用钱

操作：
1. 立刻卖出50-70%
2. 锁定亏损，保住剩余资金
3. 等3月危险期过后再考虑

数学原理：
- 现在亏27%，需涨37%才回本
- 如果再跌20%（到$68），需涨76%才回本
- 越早止损，恢复越容易
```

#### 选择B：减仓+等待（推荐给大多数人）

```
适合人群：
- 仓位<总资产10%
- 能承受继续下跌
- 不急用钱

操作：
1. 今天减仓30-50%（锁定部分损失）
2. 保留30-50%等反弹
3. 设置止损线：如果再跌10%（到$77），清仓

反弹时机观察：
- 连续3天不创新低 ✓
- 库存开始回升 ✓
- 美元指数走弱 ✓
满足2个以上再考虑补仓
```

#### 选择C：死扛等反弹（高风险）

```
适合人群：
- 仓位极小（<总资产5%）
- 完全不急用钱
- 投资期限>2年

风险警告：
⚠️ 花旗看$150 vs JPMorgan警告跌50%
⚠️ 3月还有交割+FOMC+三巫日三重风险
⚠️ 库存仍在警戒线下，随时可能再跌

操作：
1. 不加仓不摊平
2. 设置心理止损线（比如再跌20%就认亏）
3. 3月10日前必须评估是否减仓
```

## 四、突发情况应对

### 白银单日暴跌 > 10%

```
第1步：判断是否在交割窗口
  ├─ 是 → 不要抄底，等交割期结束
  └─ 否 → 可能是普通回调，观察2天

第2步：查COMEX库存
  ├─ < 1.5亿 → 继续观望，不抄底
  └─ > 1.5亿 → 可小仓试探

第3步：等企稳信号
  ├─ 连跌3天以上 → 继续等
  └─ 出现反弹 → 可分批建仓（每次10%）
```

### COMEX宕机/交易所异常

```
立刻减仓50%
不要等消息
流动性风险 = 价格剧烈波动
```

### 美联储意外鹰派

```
金银会跌
减仓30-50%
等市场消化1-2天再决定
```

### 地缘危机爆发（战争、黑天鹅）

```
⚠️ 理论可能失效
避险需求可能推高金银
不要盲目减仓
观察美元走势：
  - 美元涨 → 金银可能跌
  - 美元跌 → 金银可能涨
```

## 五、仓位管理铁律

### 最大仓位限制

| 时期 | 纸黄金最大仓位 |
|------|----------------|
| 安全期（库存>2亿，非交割） | 总资产 15% |
| 警戒期（库存1.5-2亿） | 总资产 10% |
| 高危期（库存<1.5亿或交割月） | 总资产 5% |
| 极危期（库存<1.2亿+交割窗口） | 总资产 3% 或清仓 |

### 加仓/减仓原则

| 原则 | 说明 |
|------|------|
| **分批操作** | 不要一次性全买/全卖，分3-5次 |
| **减仓要快** | 出现信号立刻减，不要等 |
| **加仓要慢** | 确认安全后分批加，不要急 |
| **留有余地** | 永远不要100%满仓或100%空仓 |

### 止损设置（重要！）

| 情况 | 止损线 | 示例 |
|------|--------|------|
| 正常持仓 | 买入价-10% | 买入$100，跌到$90止损 |
| 高危期持仓 | 买入价-5% | 更严格，减少损失 |
| 抄底试探 | 买入价-8% | 试探仓不留太多空间 |

**止损原则**：
```
1. 买入时就设好止损，不要事后补
2. 止损触发就执行，不要犹豫
3. 止损后不立刻反手，等2-3天
4. 止损是为了活下来，不是认输
```

### 心理管理

| 心理陷阱 | 症状 | 对策 |
|----------|------|------|
| **FOMO（错失恐惧）** | "涨了我没买，好后悔" | 永远有下次机会 |
| **恐慌抛售** | "跌了赶紧卖" | 看是否触发P0信号 |
| **摊平执念** | "跌了加仓拉低成本" | 顺势不摊平 |
| **回本心态** | "等回本再卖" | 止损不等人 |

**一句话**：**情绪是敌人，纪律是朋友**

## 八、踏空 vs 套牢：怎么选？

### 数据说话

| 情景 | 概率 | 后果 | 恢复难度 |
|------|------|------|----------|
| **踏空**（错过上涨） | 高 | 少赚钱 | 随时可再买 |
| **套牢**（被套住） | 低但致命 | 亏损50%+ | 需涨100%才回本 |

### 数学原理

```
亏损50% → 需要涨100%才回本
亏损30% → 需要涨43%才回本
亏损20% → 需要涨25%才回本
亏损10% → 需要涨11%才回本

结论：越早止损，恢复越容易
```

### 真实案例

| 策略 | 2026/1/30表现 | 结果 |
|------|---------------|------|
| 频繁操作（1/27减仓） | 躲过-17%暴跌 | ✅ 保住本金 |
| 长期持有（不动） | 吃满-17%跌幅 | ❌ 需涨21%回本 |

### 结论

| 适合长期持有的人 | 适合频繁操作的人 |
|------------------|------------------|
| 不看盘的 | 能每周检查的 |
| 承受得起50%回撤的 | 纪律性强的 |
| 投资期限>5年的 | 想控制风险的 |

> **本指南适合**：想控制风险、能遵守纪律的投资者
> 
> **不适合**：完全不想管的"佛系"投资者
