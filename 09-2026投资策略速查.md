| 出现这个情况 | 可以这样做 | 原因 |
|--------------|------------|------|

| FedWatch显示降息概率 > 70% | 可持有 | 政策利好 |
| 暴跌后企稳 + 库存回升 | 可分批建仓 | 风险释放 |

### 💰 获利加仓时机

| 出现这个情况 | 操作 | 原因 | 验证 |
|--------------|------|------|------|
| **暴跌>15%后连续3天不创新低** | 小仓试探（10-20%） | 超跌反弹 | ✅ 1/30暴跌后次日反弹17% |
| **FOMC鸽派指引**（暗示未来降息） | 加仓20-30% | 政策利好 | ✅ 2025/12鸽派后金银涨 |
| **美元指数<100** | 加仓 | 美元弱=金银强 | ✅ 相关系数-40%~-96% |
| **库存连续3周回升** | 逐步加仓 | 供应压力缓解 | ✅ 2024年库存充足无暴跌 |
| 交割期结束+库存回升 | 观察后建仓 | 需双重确认 | ⚠️ 单独交割结束不够 |

> ⚠️ **重要**：交割期结束≠必然反弹，分析师指出这可能是"神话"。需同时确认：库存回升+无宏观利空

### 获利原则

```
1. 分批买：不要一次满仓，分3-5次建仓
2. 等企稳：暴跌后至少等3天不再创新低
3. 设止损：买入后设5-10%止损线
4. 顺势加：涨了再加，亏了不摊平
```

## 三、突发情况应对

### 白银单日暴跌 > 10%

```
第1步：判断是否在交割窗口
  ├─ 是 → 不要抄底，等交割期结束
  └─ 否 → 可能是普通回调，观察2天

第2步：查COMEX库存
  ├─ < 1.5亿 → 继续观望，不抄底
  └─ > 1.5亿 → 可小仓试探

第3步：等企稳信号
  ├─ 连跌3天以上 → 继续等
  └─ 出现反弹 → 可分批建仓（每次10%）
```

### COMEX宕机/交易所异常

```
立刻减仓50%
不要等消息
流动性风险 = 价格剧烈波动
```

### 美联储意外鹰派

```
金银会跌
减仓30-50%
等市场消化1-2天再决定
```

### 地缘危机爆发（战争、黑天鹅）

```
⚠️ 理论可能失效
避险需求可能推高金银
不要盲目减仓
观察美元走势：
  - 美元涨 → 金银可能跌
  - 美元跌 → 金银可能涨
```

## 四、仓位管理铁律

### 最大仓位限制

| 时期 | 纸黄金最大仓位 |
|------|----------------|
| 安全期（库存>2亿，非交割） | 总资产 15% |
| 警戒期（库存1.5-2亿） | 总资产 10% |
| 高危期（库存<1.5亿或交割月） | 总资产 5% |
| 极危期（库存<1.2亿+交割窗口） | 总资产 3% 或清仓 |

### 加仓/减仓原则

| 原则 | 说明 |
|------|------|
| **分批操作** | 不要一次性全买/全卖，分3-5次 |
| **减仓要快** | 出现信号立刻减，不要等 |
| **加仓要慢** | 确认安全后分批加，不要急 |
| **留有余地** | 永远不要100%满仓或100%空仓 |
