## 📱 每日/每周检查清单

### 每天2分钟

```
□ 打开 silverdata.io
□ 记录：白银价格、黄金价格、金银比、动量
□ 填入"每日价格追踪"表
```

### 每周三5分钟（重点！）

```
□ 看 COMEX Inventory → Registered
□ 看 ETF Holdings → SLV、PSLV
□ 填入"每周三库存追踪"表
□ 对比上周变化
□ 库存↓ = 危险加剧，库存↑ = 缓解
```

### 每周五/六5分钟

```
□ 看 COT → Commercials Net、Managed Money Net
□ 填入"每周五COT追踪"表
□ 看商业是否继续REDUCED SHORT
□ 看投机是否极端做多
```

### 每周末10分钟

```
□ 填写"每周汇总表"
□ 填写"信号汇总表"
□ 数利多/利空信号
□ 决定下周操作
```
