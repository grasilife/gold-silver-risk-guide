## 📈 信号判断标准

### 各指标阈值

| 指标 | 🟢 利多 | ⚪ 中性 | 🔴 利空 |
|------|---------|---------|---------|
| **金银比** | < 50 | 50-80 | > 80 |
| **COMEX库存** | > 150M | 120-150M | < 120M |
| **动量分数** | > 70 | 30-70 | < 30 |
| **COT商业** | < -50,000 | -50,000~-20,000 | > -20,000 |

### 信号汇总判断

| 利多数 | 利空数 | 操作建议 |
|:------:|:------:|----------|
| ≥4 | ≤1 | ✅ 可加仓 |
| 3 | 2 | ⚠️ 观望 |
| 2 | 3 | ⚠️ 准备减仓 |
| ≤1 | ≥4 | 🔴 应减仓 |

## 📱 快速填写指南

### 每天2分钟

```
1. 打开 silverdata.io
2. 记录首页显示的数据：
   - Silver价格
   - Gold价格
   - Gold/Silver Ratio
   - Momentum Score
3. 周三额外看：
   - COMEX Inventory → Registered
4. 周五额外看：
   - COT → Commercials Net
5. 数信号，填"信号"列
```

### 数据位置

| 数据 | 在页面哪里 |
|------|------------|
| 白银价格 | 顶部左侧 "Silver (SI=F)" |
| 黄金价格 | 顶部右侧 "Gold (GC=F)" |
| 金银比 | "Gold/Silver Ratio" 卡片 |
| 动量分数 | "Momentum Score" 卡片 |
| COMEX库存 | "COMEX Inventory" → Registered |
| COT商业 | "COT Positioning Details" → Commercials Net |
