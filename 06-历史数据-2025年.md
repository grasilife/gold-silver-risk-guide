# 2025年贵金属市场回顾（危机爆发年）

> 2025年是COMEX白银挤兑危机爆发的一年，库存跌破警戒线，首次出现大规模交割恐慌

---

## 一、2025年核心结论

- ✅ **首次大规模白银挤兑发生在2025年10月**（非2024年）
- 黄金突破$3,000里程碑（3月14日）
- COMEX白银库存从2.3亿降至1.27亿盎司
- 12月交割潮：4760万盎司在4天内被提取

### 关键时间线

### 里程碑意义
- 从$2,500到$3,000仅用**210天**

1. 中东局势紧张
2. 特朗普关税政策
3. 央行持续购金（尤其中国）
4. 美联储降息预期
5. 去美元化趋势

## 四、2025年10月白银挤兑（首次大规模危机）

### 事件背景

| 因素 | 状态 |
|------|------|
| COMEX注册库存 | ~1.5亿盎司（接近警戒线） |
| 伦敦白银租借利率 | **39%**（正常的7倍） |
| 库存流出 | 单日最大流出（4年来） |
| 宏观背景 | 美国联邦政府关门（10月5-10日）+ 关税升级 |

### 价格暴跌详情

| 日期 | 价格 | 涨跌幅 | 说明 |
|------|------|--------|------|
| 10月16日 | $53.02 | - | 52周高点 |
| 10月17日 | ~$51 | -4% | 开始回调 |
| 10月20日 | ~$50 | -2% | 继续下跌 |
| **10月21日** | **$47.45** | **-7.18%** | **单日最大美元跌幅（自2011年9月）** |

### 暴跌原因
1. **库存接近警戒线**：1.5亿盎司逼近临界点
2. **租借利率飙升**：伦敦市场白银严重短缺
3. **政府关门**：市场避险情绪上升
4. **获利了结**：从$29涨至$53后回调

### 恢复情况
- 暴跌后约**2周**企稳
- 但未能回到$50以上
- 年末继续下探至$28.94

## 五、2025年12月交割潮

### 核心数据

| 指标 | 数据 |
|------|------|
| 12月交割量 | **4,760万盎司** |
| 提取时间 | **4天** |
| 占注册库存比例 | **60%+** |
| 交割后库存 | ~1.27亿盎司 |

### 为什么12月交割量如此大？

1. **伦敦市场枯竭**：机构转向COMEX要求实物
2. **年末结算需求**：基金/机构年底清算
3. **库存紧张预期**：提前囤积实物
4. **3月合约担忧**：为2026年3月交割做准备

### 市场影响
- 白银从$45区间跌至$28.94
- 触发2026年1月的暴涨（从$29到$120）
