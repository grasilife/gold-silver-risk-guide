## 突发情况应对

### 白银单日暴跌 > 10%

```
1. 判断是否在交割窗口
   ├─ 是 → 不要抄底，等交割期结束
   └─ 否 → 可能是普通回调，观察2天

2. 查COMEX库存（silverdata.io）
   ├─ < 1.5亿 → 继续观望，不抄底
   └─ > 1.5亿 → 可小仓试探
```

### COMEX宕机/交易所异常

```
立刻减仓50%，不要等消息
```

### 美联储意外鹰派

```
减仓30-50%，等市场消化1-2天
```

## 分层优先级体系（P0→P1→P2→P3）

| 层级 | 指标 | 作用 | 查询 |
|------|------|------|------|
| **P0** | 库存<1.5亿 + 交割窗口 | **必要条件**，都不满足=安全 | silverdata.io |
| **P1** | 逆价差、涨幅>50%、CFTC多头减少 | **放大器**，决定跌多少 | tradingcharts.com |
| **P2** | FOMC、非农、CPI、三巫日 | **催化剂**，决定什么时候跌 | 本文档日历 |
| **P3** | ETF持仓、租借利率、金银比 | **确认信号** | kitco.com |

**判断逻辑**：P0不触发=安心 → P0触发=看P1定减仓比例 → P2定时间点 → P3确认
